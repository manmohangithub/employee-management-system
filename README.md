🚀 Employee Management System

Full-stack web app to manage employee records with React.js, Spring Boot, and MySQL.

⚡ Key Features
Add, update, delete employees
View employee list
REST API integration
Responsive UI
🧠 Tech Stack

React.js | Spring Boot | MySQL

⚙️ Run Locally
# Backend
cd backend
mvn spring-boot:run

# Frontend
cd frontend
npm install
npm start
🔌 APIs

GET /employees
POST /employees
PUT /employees/{id}
DELETE /employees/{id}
